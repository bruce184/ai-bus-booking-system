# Agent Task Playbook

## Before Coding

1. Inspect local repo state.
2. Read `AGENTS.md`.
3. Read `docs/agent-context/00-index.md`.
4. Read `docs/DEV_WORKFLOW.md`.
5. Read `docs/CODING_GUIDELINES.md`.
6. Read the assignment row.
7. Read task-specific source-of-truth docs.

## While Coding

1. Change only assigned files/area.
2. Keep contract files updated with code behavior.
3. Add focused tests or manual verification notes when available.
4. Do not commit secrets or real data.
5. Record out-of-scope findings.
